<Callout type="warning" title="The package is not published">
  `liveryscript` is not currently available from the public npm registry. Use the repository until release status changes.
</Callout>

```sh
git clone https://github.com/jerkeyray/livery.git
cd livery
bun install
bun run build
```

Livery requires Node 22 or newer and uses Bun workspaces for development.

## Try the playground

```sh
bun run dev
```

The Vite playground is the fastest local feedback loop for source, diagnostics, themes, responsive widths, and timeline states.

## Use the built distribution

Until publication, use the built `packages/liveryscript` distribution from a pinned checkout. It contains the root, `/web`, `/react`, `/node`, stylesheet, and `livery` CLI entry points as one versioned artifact. Do not install the private implementation workspaces directly.

After publication is verified, the install commands will be:

```sh
bun add liveryscript
bun add liveryscript @resvg/resvg-js # only for PNG
```

## Verify the checkout

```sh
bun run typecheck
bun run test
bun run build
```
