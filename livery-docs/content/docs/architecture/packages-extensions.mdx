Core owns language, semantics, solving, validation, themes, timelines, catalogs, diagnostics, and portable exports. Web adds DOM retention. React wraps Web. Export Node rasterizes Core SVG. CLI composes Core and Export Node. These are private workspace boundaries bundled behind `liveryscript`; consumers install only that package. The docs site bootstraps the same facade and hosts Studio.

## Supported extension points

- Authored source `component` declarations.
- `LiveryTheme` and bounded token overrides.
- Trusted application-owned `IconRegistry` values.
- Exact remote-image host allowlists and data-image limits.
- Documented measurement and render options.

## Internal or compatibility surfaces

Do not build new integrations around legacy artifacts, generic headless layout adapters, deprecated direct instance getters, solver candidate internals, or undocumented parser structures. They remain exported where compatibility or tooling requires them, but the `figure` вҶ’ `VisualDocument` вҶ’ `BoardScene` pipeline is canonical.
