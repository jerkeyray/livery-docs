---
title: Exports and CLI
description: Produce deterministic SVG, JSON, and PNG from the same solved scene.
---

React, the browser runtime, SVG, JSON, PNG, and the CLI all start from the same compiled document and solved geometry. Export is not a separate drawing implementation.

## SVG

```ts
import { exportVisual, paperTheme } from "@jerkeyray/core";

const result = exportVisual(source, {
  format: "svg",
  theme: paperTheme,
  width: 720,
});

if (!result.output) {
  throw new Error(result.diagnostics.map((item) => item.message).join("\n"));
}

await Bun.write("figure.svg", result.output);
```

The SVG includes an accessible title and description derived from the figure and solved scene.

## Scene JSON

```ts
const result = exportVisual(source, {
  format: "json",
  width: 720,
  pretty: true,
});
```

JSON includes the compiled document, diagnostics, solved scene, validation report, and layout attempts when available. It is useful for snapshots, debugging, and downstream tooling.

## PNG in Node or Bun

```ts
import { writeFile } from "node:fs/promises";
import { exportVisualPng } from "@jerkeyray/export-node";

const result = exportVisualPng(source, {
  width: 720,
  scale: 2,
  background: "#ffffff",
});

if (!result.output) {
  throw new Error(result.diagnostics.map((item) => item.message).join("\n"));
}

await writeFile("figure.png", result.output);
```

PNG export uses Resvg and returns `Uint8Array`. Use `outputWidth` when the raster width matters more than a scale factor.

## CLI

The CLI accepts a file or stdin and infers the format from the output extension:

```sh
livery diagram.livery --output diagram.svg
livery diagram.livery --output diagram.png --scale 2
livery diagram.livery --output diagram.svg --theme midnight
cat diagram.livery | livery - --format json --pretty
```

Explicit flags are useful in CI:

```sh
livery diagram.livery \
  --format svg \
  --width 720 \
  --output diagram.svg
```

The process reports diagnostics to stderr and exits unsuccessfully when no requested output can be produced.

`--theme` accepts `editorial`, `paper`, `midnight`, `blackout`, `blueprint`, or `monochrome`. Programmatic exports additionally accept a custom `LiveryTheme` and trusted `IconRegistry`.

## Responsive export is multiple solves

An SVG has one solved width. If you need chat, article, and expanded variants, render each width deliberately:

```ts
for (const width of [360, 720, 1200]) {
  const result = exportVisual(source, { format: "svg", width });
  if (result.output) await Bun.write(`figure-${width}.svg`, result.output);
}
```

The source stays the same; the pinboard solve changes with the available width.
