title: The Livery language
description: One bounded visual language for human authors and generating agents.
---

A Livery file declares reusable components and exactly one canonical `figure`. Inside it, bindings create stable identities, connectors relate anchored elements, and one root layout describes the composition.

```livery
figure checkout("Checkout") {
  customer = person("Customer")
  api = service("Checkout API")
  payment = service("Payment provider")

  submit = customer.right -> api.left("submit")
  authorize = api.right -> payment.left("authorize")

  row(customer, api, payment, gap: xl)
}
```
